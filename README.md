# Ishika Pancholi — Portfolio

Personal portfolio site built from scratch using HTML, CSS, and JavaScript.

**Live site:** [ishikapancholi.netlify.app](https://ishikapancholi.netlify.app)

## Built With
- HTML5 & CSS3 — custom layout, animations, and responsive design
- Vanilla JavaScript — interactive Chart.js dashboards, smooth scroll, mobile nav
- Chart.js — data visualizations for each project card
- Google Fonts — Cormorant Garamond + Outfit
- Netlify — continuous deployment from private GitHub repo

## Features
- Responsive design with mobile hamburger navigation
- Interactive data dashboards embedded in project cards
- Animated marquee, fade-in sections, and scroll effects
- Certifications and education timeline
- Contact section with direct links

## Projects Featured
1. **S&P 500 Stock Market Volatility & Risk Analysis** — Python, Pandas, Tableau
2. **MarketIQ — DA Career Market Analysis** — Python, scikit-learn, statsmodels
3. **SEC EDGAR Retail Financial Data Pipeline** — Python, PostgreSQL, ETL
